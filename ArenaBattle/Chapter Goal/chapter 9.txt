//// 콜리전의 3가지 방법 ////
- 스태틱메시 에셋 ; 스태틱메시 애셋에 콜리전 영역을 심는 방법. 
			스태틱메시를 더블 클릭하면 열리는 '스태틱메시 에디터'에서 해당 콜리전 영역을 설정하고 확인
			스태틱메시 컴포넌트에서 비쥬얼과 충돌, 두 가지 기능을 설정할 수 있다



-기본 도형(Primitive) 컴포넌트 ; 기본 도형을 이용하여 충돌영역을 지정하는 방법
				스태틱메시와 별도로 충돌 영역을 제작하는데 사용. 스켈레탈 메시를 움직일 때 사용


-피직스 애셋; 특정 상황에서 Rag Doll효과를 구현할 때 사용. 
	캐릭터의 각 부위에 기본 도형으로 충돌영역을 설정하고 이를 연결해 캐릭터의 물리를 설정
	스켈레탈 메시에서만 사용




///물리 설정의 3가지 ////

1) 콜리전 채널과 기본 반응
2) 콜리전 채널의 용도
3) 다른 콜리전 채널과의 반응

////UE의 기본 콜리전 채널 /////

- WorldStatic :  움직이지 않는 정적인 배경 액터에 사용한다
		static mesh -> static component
		
- WorldDynamic : 움직이는 액터에 사용
		blueprint -> static component 이용


- Pawn : 플레이어가 조종하는 물체
	capsul component
	
- Visibility : 배경 물체가 시각적으로 보이는지 탐지하는 데 사용
		탐지에서 Pawn은 제외. 
		마우스로 물체를 선택하는 picking을 구현할 때 사용 

- Camera : 카메라와 목표물 간에 장애물이 있는지 탐색하는데 사용
	GTA 방식 카메라에서 장애물이 시야를 가리면 
	카메라를 장애물 앞으로 줌인하는 기능

- PhysicsBody : 물리 시뮬레이션으로 움직이는 컴포넌트 설정
